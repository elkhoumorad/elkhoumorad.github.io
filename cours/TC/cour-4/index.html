Le Modèle de l'Atome

👨‍🏫 Cours rédigé par Morad | Niveau : Tronc Commun Scientifique

1. De quoi est fait un atome ?

L'atome est la brique de base de la matière. Bien qu'il soit infiniment petit, il n'est pas "plein". Il a une structure lacunaire (il est fait à 99,99% de vide !). Il est composé de deux parties principales :

  • Un noyau central : Très dense et chargé positivement, il contient des particules appelées nucléons (les protons $\oplus$ et les neutrons sans charge).
  • Des électrons ($\ominus$) : Particules extrêmement légères et chargées négativement qui tournent à très grande vitesse autour du noyau.
💡 La règle de neutralité :

Dans la nature, un atome est toujours électriquement neutre. Cela veut dire qu'il possède toujours exactement le même nombre de protons (charges $+$) que d'électrons (charges $-$) !

2. Le symbole du noyau et les Isotopes

Pour décrire la carte d'identité du noyau d'un élément chimique, on utilise une écriture standardisée :

Le noyau d'un atome $\text{X}$ se note :

$$_{Z}^{A} \text{X}$$
  • $X$ : Le symbole chimique de l'élément (ex: $\text{C}$ pour Carbone, $\text{Fe}$ pour Fer).
  • $A$ (Nombre de masse) : Le nombre total de nucléons (protons + neutrons) dans le noyau.
  • $Z$ (Numéro atomique) : Le nombre de protons (qui est aussi égal au nombre d'électrons puisque l'atome est neutre).

Pour calculer le nombre de neutrons $N$, il suffit de faire une petite soustraction :

$$N = A - Z$$
📌 Que sont les Isotopes ?

Des atomes sont dits isotopes s'ils ont le même numéro atomique $Z$ (donc c'est le même élément chimique), mais un nombre de masse $A$ différent (ils n'ont pas le même nombre de neutrons).
Exemple : Le Carbone $12$ ($_{6}^{12}\text{C}$) et le Carbone $14$ ($_{6}^{14}\text{C}$).

3. Le cortège électronique (Les couches K, L, M)

Les électrons ne tournent pas n'importe où ! Ils se répartissent sur des niveaux d'énergie bien précis que l'on appelle couches électroniques, notées K, L, M, N...

Chaque couche ne peut contenir qu'un nombre maximum d'électrons, donné par la formule $2n^2$ (où $n$ est le numéro de la couche) :

  • Couche K ($n=1$) : Maximum 2 électrons.
  • Couche L ($n=2$) : Maximum 8 électrons.
  • Couche M ($n=3$) : Maximum 18 électrons (mais on s'arrête souvent à 8 pour les éléments du Tronc Commun !).

Exemple de configuration électronique :

Pour l'atome d'Oxygène ($_{8}^{16}\text{O}$), on a $Z=8$, donc 8 électrons à ranger :

$$\text{Configuration : } (K)^2 (L)^6$$

Les électrons situés sur la toute dernière couche remplie sont appelés électrons de valence (ou périphériques). Ce sont eux qui vont participer aux réactions chimiques !
